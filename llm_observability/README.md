# llm_observability
# Overview
Repository for Stress test with k6 and Datadog \
blog link: https://zenn.dev/hosimesi/articles/fbdf2083c883b0


# How to use
1. Install uv.
    ```bash
    $ brew install uv
    ```
2. Copy .env.example to .env.
   ```bash
   $ cp .env.example .env
   ```
3. Set Secret Key to .env.
4. Run.
    ```bash
    $ make run
    ```
